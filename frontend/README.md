# Loomaxis Studio Frontend

The frontend is a custom React + Vite experience built to feel like a premium deployment product instead of a utilitarian admin form.

## Run

```bash
npm install
cp .env.example .env
npm run dev
```

## Build

```bash
npm run build
```

## Environment

- `VITE_BACKEND_URL`: Base URL for the backend API. Defaults to `http://localhost:8000`.

## Highlights

- Command-deck style hero and deployment workspace
- File, folder, and drag-and-drop BPMN staging
- Bundled sample workflow for quick demos
- Responsive layout tuned for desktop and laptop operations use
